## 📦 TEMPLATES A CREAR (10 templates base)

### 1. TEMPLATE "DATO IMPACTANTE"
```
[FONDO: Degradado oscuro (#1A1A2E → #0F3460)]
[CENTRO: Número grande en dorado (#D4AF37) - ej: "144%"]
[DEBAJO: Texto breve en blanco - ej: "El oro en 5 años"]
[ABAJO: Tu logo/nombre + CTA - "Escríbeme ORO"]
[DECORACIÓN: Sutil brillo dorado en esquinas]
```

### 2. TEMPLATE "COMPARATIVA"
```
[FONDO: Blanco]
[DIVIDIDO EN 2 COLUMNAS:]
  Izquierda (rojo): ❌ BANCO
    - Dato 1
    - Dato 2
    - Dato 3
  Derecha (verde): ✅ ORO
    - Dato 1
    - Dato 2
    - Dato 3
[ABAJO: CTA dorado]
```

### 3. TEMPLATE "PASO A PASO"
```
[FONDO: Gris claro (#F8F9FA)]
[TÍTULO arriba en oscuro]
[3 CAJAS con números 1, 2, 3 en círculos dorados]
[Cada caja: icono + texto breve]
[ABAJO: CTA]
```

### 4. TEMPLATE "CARRUSEL" (10 slides)
```
Slide 1: Título gancho (fondo oscuro, texto dorado)
Slides 2-9: Contenido (fondo blanco, texto oscuro, acentos dorados)
Slide 10: CTA final (fondo dorado, texto oscuro "Escríbeme ORO")
```

### 5. TEMPLATE "PREGUNTA/RESPUESTA"
```
[ARRIBA: Pregunta en grande con ❓]
[MEDIO: Respuesta en texto más pequeño]
[ABAJO: Tu marca + CTA]
[ESTILO: Fondo oscuro, texto blanco, acentos dorados]
```

### 6. TEMPLATE "STORY ENCUESTA"
```
[FONDO: Degradado oscuro/dorado]
[TEXTO: Pregunta en grande]
[ESPACIO: Para poner sticker de encuesta de Instagram]
[ESTILO: Minimalista, impactante]
```

### 7. TEMPLATE "TESTIMONIO"
```
[FONDO: Blanco con borde dorado sutil]
[ARRIBA: Comillas grandes doradas " "]
[CENTRO: Texto del testimonio]
[ABAJO: Nombre + foto (o avatar)]
[ESTRELLAS: ⭐⭐⭐⭐⭐]
```

### 8. TEMPLATE "PRECIO DEL DÍA"
```
[FONDO: Oscuro]
[ARRIBA: "PRECIO DEL ORO HOY"]
[CENTRO: "116€/gramo" en grande dorado]
[ABAJO: Flecha arriba/abajo + variación %]
[PIE: "Hace 5 años: 48€ | +144%"]
```

### 9. TEMPLATE "MITO DESMONTADO"
```
[FONDO: Blanco]
[ARRIBA: "MITO:" + texto tachado en rojo]
[MEDIO: "REALIDAD:" + texto en verde]
[ABAJO: CTA]
[ESTILO: Limpio, directo, contrastante]
```

### 10. TEMPLATE "CTA FINAL"
```
[FONDO: Dorado degradado]
[CENTRO: Texto directo "¿Listo para empezar?"]
[DEBAJO: "Escríbeme ORO por DM" con icono de mensaje]
[ABAJO: Tu nombre/perfil]
[ESTILO: Llamativo, urgente pero elegante]
```
